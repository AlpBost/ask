##### \# ask - LLM CLI Tool

A simple Bash script to interact with LLMs via command line.



###### \## Prerequisites

\* \[cite\_start]curl 

\* \[cite\_start]jq 



###### \## Configuration

Export these variables:

export ASK\_API\_URL="your\_url"

export ASK\_MODEL="your\_model"

export ASK\_API\_KEY="your\_key"



###### \## Usage Examples 

1\. `ask "Hello world"`

2\. `cat file.txt | ask "Explain this"` 



###### \## Limitations 

\* Requires internet connection.

\* Only works with OpenAI-compatible APIs.
